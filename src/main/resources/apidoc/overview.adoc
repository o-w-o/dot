[[overview-http-verbs]]
=== HTTP 动词

API 会尽量遵循标准的 HTTP 和 REST 约定使用 *HTTP 动词* 。

|===
| Verb | 说明

| `GET`
| 获取

| `POST`
| 创建

| `PATCH`
| 更新(部分或全部)

| `DELETE`
| 删除
|===

[[overview-http-status-codes]]
=== HTTP 状态码

API 会尝试尽可能地遵守标准 HTTP 和 REST 约定的 *HTTP 状态代码* 。

|===
| Status code | 说明

| `200 OK`
| 客户端请求完成

| `400 Bad Request`
| 客户端请求异常

| `401 Unauthorized`
| 未授权

| `403 Forbidden`
| 授权不足

| `404 Not Found`
| 不存在

| `500 Internal Server Error`
| 服务端异常
|===
